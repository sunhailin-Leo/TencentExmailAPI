<h3 id="MostEssential">必看点</h3>

* 该API需要企业邮箱管理员才能进行开发(涉及唯一的CorpID和不同接口对应不同的secret码), 因此普通的开发者无法获取。
* 需要开发阅读过企业邮箱的开发者文档 [地址](https://exmail.qq.com/qy_mng_logic/doc#10003)

---

<h3 id="DevEnv">开发环境</h3>

* 系统版本: Windows 10 x64
* Python版本: Python 3.4.4 x64
